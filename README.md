# Victim Advocate — AI-Powered Crime Victim Rights Engine

**Know your rights. Immediately. In your state.**

## What It Does

Victim Advocate provides instant, state-specific legal rights information for crime victims. No lawyer required to understand your basic rights — get them in plain English, right now.

## Coverage

### States (9)
Florida, Georgia, Illinois, Michigan, New York, North Carolina, Ohio, Pennsylvania, Texas + Western states

### Crime Types (7)
Domestic Violence, Sexual Assault, Stalking, Child Abuse, Human Trafficking, Robbery/Assault, Identity Theft/Fraud

## Features

- **Instant rights lookup** — select your state and crime type
- **Plain English explanations** — no legal jargon
- **PDF report generation** — downloadable rights summary
- **Resource directory** — hotlines, shelters, legal aid by state
- **3-tier pricing** — Free (basic lookup), Pro ($12/mo full reports), Enterprise (API/org access)
- **No data collection** — we don't track what you search

## Tech Stack

- Flask + Gunicorn
- State rights modules (pure Python, no external API needed)
- PDF generation
- Stripe payments integration

## Deployment

Configured for Railway deployment:
```
railway up
```

## Why This Matters

Every year millions of crime victims don't know their rights. They don't know about compensation funds, protection orders, or victim notification systems that exist in their state. This app fixes that.
